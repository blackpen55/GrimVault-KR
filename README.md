# GrimVault-KR 배포 안내

## GrimVault 한글 클라이언트 지원 및 실제 경매장 가격 반영 기능 추가

Dark and Darker 시세 측정 프로그램인 **GrimVault**를 한국어 클라이언트에서도 사용할 수 있도록 수정한 버전입니다.

기존 GrimVault는 DarkerDB에서 제공하는 API를 기반으로 동작하며, 본 버전은 원본 기능을 바탕으로 한국어 클라이언트용 인식 및 변환 과정을 추가했습니다.
또한 동일한 옵션을 가진 아이템의 실제 경매장 가격을 우선적으로 반영하도록 가격 측정 로직을 일부 추가했습니다.

원작자가 SDF 측에 문의했을 때 사용에 문제가 없는 프로그램이라고 안내받은 것으로 알고 있으며, GrimVault 또한 MIT 라이선스로 공개되어 있어 한국어 클라이언트용 패치 버전으로 배포합니다.
다만 문제가 있을 경우 글은 삭제하겠습니다.

## 주요 기능

* DarkerDB 기준 시세 조회
* 한국어 클라이언트 아이템 정보 인식 지원
* 동일 아이템, 동일 등급, 동일 옵션 수치의 경매장 가격 우선 표시
* 최근 판매 기록 및 현재 등록 매물을 기반으로 한 가격 보정

---

## 설치 방법

1. 한글판 다운로드 링크로 이동합니다.
2. Assets 항목에서 `GrimVault-KR-portable-0.2.0.zip` 파일을 다운로드합니다.
3. 압축을 해제합니다.
4. 폴더 안의 `GrimVault-KR.exe`를 실행합니다.
5. Windows 보안 경고가 표시될 경우 `추가 정보` → `실행`을 선택합니다.
6. 실행 후 `F1` 키를 누르면 현재 버전을 확인할 수 있습니다.

---

## 사용 방법

### 모드 전환

`F6` 키를 눌러 아래 모드를 전환할 수 있습니다.

* 자동 모드
* 수동 모드
* 비활성화 모드

### 수동 모드 사용법

1. 아이템 위에 마우스를 올립니다.
2. `F5` 키를 누릅니다.
3. 시세 정보를 확인합니다.

처음 측정할 때는 로딩이 다소 걸릴 수 있습니다.
정상적으로 표시되지 않을 경우, 아이템 위에 마우스를 올린 상태에서 `F5`를 다시 눌러보시기 바랍니다.

## 참고 사항

본 버전은 원본 GrimVault에 다음 과정을 추가한 형태입니다.

* 이미지 기반 한국어 아이템 정보 인식
* 한국어 텍스트 → 영어 데이터 변환
* 실제 경매장 가격 반영 로직

이 과정이 추가되어 프로그램 용량은 약 400MB 정도입니다.

실제 경매장 가격이나 시세와 차이가 있을 수 있으므로, 표시되는 가격은 참고용으로만 사용해주시기 바랍니다.
현재 일부 오류가 있을 수 있으며, 확인되는 문제는 계속 수정할 예정입니다.

---

## 가격 측정 로직

* **시장가**: 최근 7일 판매 완료 매물 중 옵션 수치까지 같은 매물의 중앙값
* **동일매물가**: 현재 판매 중인 매물 중 옵션 수치까지 같은 매물의 최저가
* **유사매물가**: 현재 판매 중인 유사 옵션 매물의 최저가

---

## 오류 제보 안내

오류 제보 시에는 아래 정보를 함께 남겨주시면 수정에 도움이 됩니다.

* 아이템 이름
* 오류창 내용
* 문제가 발생한 상황
* 가능하다면 전체 화면 스크린샷

예시 형식:

`GrimVault 버그) 아이템 이름 / 오류 내용 / 발생 상황`

스크린샷까지 함께 올려주시면 원인 파악이 더 쉽습니다.

## 수정 사항

### 0.1.1

* `F5` 입력 시 무응답 상태가 발생하던 경우, 오류 메시지가 출력되도록 수정했습니다.
* 기존 버전이 정상적으로 작동하는 사용자는 새로 설치할 필요는 없습니다.

### 0.1.2

* Visual C++ 관련 설치 문제를 해결했습니다.
* 업데이트 후에는 트레이에서 GrimVault를 완전히 종료한 뒤 다시 실행해주시기 바랍니다.
* 현재 정상적으로 작동하는 사용자는 새로 설치하지 않아도 됩니다.
### 0.2.0

* 가격 검색 속도를 개선했습니다.
* 가격 측정 로직을 수정했습니다.
* 최신 목록 기준으로 한국어 아이템 매핑을 보완했습니다.

### 0.3.0

* 트레이에서 새 버전을 확인하고 바로 업데이트할 수 있습니다.
* 새 버전이 있으면 트레이 아이콘에 알림 표시가 나타납니다.
* 업데이트 진행창을 숨겨도 트레이에서 다시 열 수 있습니다.
* 업데이트 중 남을 수 있던 OCR 프로세스 누적 문제를 줄였습니다.
* 트레이 메뉴에서 업데이트 내용을 바로 확인할 수 있습니다.
