# Third-Party Notices

GrimVault-KR is an unofficial Korean client build derived from GrimVault. The upstream project license is included in `LICENSE`.

## Fonts

The packaged Korean UI uses `assets/fonts/NotoSansKR-Regular.ttf`.

Noto Sans KR is part of the Noto font family and is licensed under the SIL Open Font License 1.1. The license text is included at `assets/fonts/OFL.txt`.

Unused bundled fonts with unclear or unnecessary distribution status were removed from this repository and from the portable release package.

## Services

Market price requests are sent to the DarkerDB API at `https://api.darkerdb.com`. GrimVault-KR does not bundle DarkerDB credentials.

Dark and Darker names, icons, and visual style references belong to their respective owners. GrimVault-KR is an unofficial community tool.
