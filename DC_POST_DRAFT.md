# DCInside Post Draft

## 제목

시세측정 확장프로그램 GrimVault 한글패치 배포함

## 본문

나다.

다크앤다커 시세측정 확장프로그램 한글패치했다.

원본은 GrimVault고 DarkerDB 기반으로 돌아감. 원 제작자 형님들한테 무한한 존경 보낸다. 형님 형님 형님.

원래 GrimVault 자체는 sdf한테 물어봐서 사용해도 되는 프로그램이라고 들었다. 그래도 이건 내가 한국어판으로 손댄 비공식 한글패치라서, 혹시 문제 있으면 바로 자삭함.

다운로드:
https://github.com/blackpen55/GrimVault-KR/releases/latest

원본:
https://darkerdb.com/

뭐 하는 프로그램이냐면 아이템에 마우스 올리고 F5 누르면 DarkerDB 기준 시세를 보여주는 프로그램임.

이번 한글패치판은 한국어 클라이언트 툴팁을 OCR로 읽고, 아이템 이름/등급/랜덤 옵션을 영어로 바꿔서 원래 GrimVault처럼 DarkerDB에 보내는 방식임.

되는 것:
- 한국어 아이템 이름 인식
- 한국어 랜덤 옵션 인식
- 등급 반영
- 시장가/상점가/칸당 가치 표시
- OCR 실패하면 인식된 이름, 추정 등급, OCR 원문 표시

아직 테스트판이라 오류 뜰 수 있음.

오류 뜨면 댓글에 이렇게 알려줘라:
- 어떤 상황에서 썼는지
- 어떤 아이템에 썼는지
- 오류창에 뭐라고 떴는지
- 가능하면 아이템 툴팁 스샷

특히 아이템 이름을 이상하게 읽는 경우가 있을 수 있음. 그런 건 보정 매핑 추가하면 고칠 수 있음.

다운로드/실행 방법:
1. 위 GitHub 링크 들어감
2. Releases에서 최신 버전 클릭
3. Assets 아래에서 `GrimVault-KR-0.1.0-portable.zip` 다운로드
4. 압축 풀기
5. `GrimVault-KR.exe` 실행
6. 다크앤다커 켜고 아이템에 마우스 올림
7. F5 누르면 시세창 뜸

주의사항:
- 처음 실행이나 게임 캡처가 안 되면 관리자 권한으로 실행해봐라
- 백신이 처음 보는 exe라 경고할 수 있음
- 아직 완성판 아님. 제보 받으면서 고칠 예정
- 원본 GrimVault랑 별개인 비공식 한글패치임

잘 되면 쓰고, 안 되면 댓글로 증상 알려줘라.
