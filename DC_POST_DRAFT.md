# 제목

시세측정 확장프로그램 GrimVault 한글패치 테스트판 배포함

# 본문

나다.

다크앤다커 시세측정 확장프로그램 GrimVault 한글패치했다.

먼저 원본은 GrimVault고, 기반 서비스는 DarkerDB다. 원본 만든 형님들한테 존나 무한한 감사와 존경을 표한다. 진짜 형님 형님 형님이다.

원본 홈페이지:
https://darkerdb.com/

GrimVault-KR 깃헙:
https://github.com/blackpen55/GrimVault-KR

다운로드:
https://github.com/blackpen55/GrimVault-KR/releases/latest

일단 이 프로그램 자체는 원래 GrimVault고, sdf한테 물어봤을 때 사용해도 괜찮은 프로그램이라고 했다. 그리고 원본 라이선스도 MIT 라이선스라서 한글 클라이언트에서 쓸 수 있게 패치해서 올려봄.

그래도 혹시 문제 있으면 바로 자삭함.

## 이게 뭐 하는 프로그램이냐

아이템에 마우스 올리고 F5 누르면 DarkerDB 기준으로 시세를 보여주는 프로그램임.

원래 GrimVault는 영어 클라이언트 기준이라 한글 클라에서는 아이템 이름이랑 옵션을 제대로 못 읽는다. 그래서 이 버전은 한글 툴팁을 OCR로 읽고, 아이템 이름/등급/랜덤 옵션을 영어로 바꿔서 원래 GrimVault처럼 DarkerDB에 조회하는 방식이다.

## 되는 것

- 한글 아이템 이름 인식
- 한글 랜덤 옵션 인식
- 아이템 등급 반영
- 장비류 가격 조회
- 팔찌 같은 잡템/소위 폐지류 일부 가격 조회
- OCR 실패하거나 매핑 실패하면 무슨 이름으로 인식했는지 오류창에 표시
- 추가 무기 공격력 같은 옵션도 가격 반영

아직 테스트판이라 오류 있을 수 있음. 특히 OCR이 아이템 이름을 이상하게 읽는 경우가 있을 수 있는데, 그런 건 매핑 추가하면 고칠 수 있다.

## 설치 방법

GitHub 처음 쓰는 사람 기준으로 적음.

1. 아래 링크로 들어간다.

https://github.com/blackpen55/GrimVault-KR/releases/latest

2. 페이지 아래쪽에 `Assets`라는 부분을 찾는다.

3. `GrimVault-KR-0.1.0-portable.zip`을 누른다.

4. zip 파일이 다운로드되면 압축을 푼다.

5. 압축 푼 폴더 안에 있는 `GrimVault-KR.exe`를 더블클릭한다.

6. 윈도우 보안 경고가 뜨면 `추가 정보` 누르고 `실행` 누르면 된다.

7. 다크앤다커를 켠다.

8. 인벤토리에서 아이템에 마우스를 올린다.

9. F5를 누르면 시세창이 뜬다.

## 사용법

- 아이템 위에 마우스 올림
- F5 누름
- 시세창 확인

안 뜨면 일단 아이템 툴팁이 화면에 제대로 보이는지 확인해라. 툴팁이 잘려 있거나 너무 빨리 움직이면 OCR이 못 읽을 수 있음.

## 오류 제보할 때

오류 나면 그냥 “안 됨”만 쓰지 말고 아래처럼 알려주면 고치기 훨씬 쉽다.

- 어떤 상황에서 눌렀는지
- 어떤 아이템에 썼는지
- 오류창에 뭐라고 떴는지
- 가능하면 아이템 툴팁 스샷
- 가능하면 오류창 스샷

예를 들면 “리벳이 달린 장갑에 F5 눌렀는데 인식 이름이 이상하게 뜸” 이런 식이면 바로 매핑 추가해서 고칠 수 있음.

## 주의사항

- 테스트판임
- 처음 실행하면 백신이나 윈도우가 처음 보는 exe라 경고할 수 있음
- 설치형이 아니라 portable 버전이라 압축 풀고 exe 실행하면 됨
- 압축 푼 폴더 안 파일을 일부만 빼서 실행하지 말고 폴더째로 둬라
- 원본 GrimVault와 DarkerDB 제작자들에게 다시 한 번 무한한 감사를 표함

문제 있으면 말해라. 진짜 문제 있으면 자삭함.
